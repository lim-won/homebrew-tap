# lim-won/homebrew-tap

Homebrew tap for [cmux-iphone](https://github.com/lim-won/cmux-iphone) — bridge
your Claude Code / Codex / cmux sessions to your iPhone & Apple Watch.

```sh
brew install lim-won/tap/cmux-iphone
cmux-iphone setup
```

The formula tracks each release of the main repo; `url` + `sha256` are updated
per tag.
